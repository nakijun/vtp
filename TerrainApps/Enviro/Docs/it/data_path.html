Enviro

Il Percorso Dati

  • Per posizionare la cartella Dati in un'altra collocazione, oppure in diverse collocazioni all'interno del computer, si deve modificare il Percorso Dati.
  • Il modo più facile per questa operazione è utilizzare la Finestra di Dialogo Gestione Terreno.

Il file vtp.xml

  • In alternativa, si può modificare il file vtp.xml. Tale file viene creato (se non esiste) la prima volta che si lancia Enviro.
    Questo è collocato nella cartella delle impostazioni.
    Su Windows, il percorso tipico è C:\Documents and Settings\Utente\Dati applicazioni
    Su Unix, il percorso tipico è la cartella home del proprio account utente.
  • Se i dati si trovano in posti differenti, si possono semplicemente inserire diversi elementi "DataPath" all'interno del file vtp.xml, e tutti i dati su tutti i percorsi saranno disponibili per tutte le appilcazioni VTP.
  • Un percorso può essere relativo o assoluto, ed è preferibile che termini con il simbolo dello slash "/". Ad esempio, il percorso di default è il seguente:
        <DataPath>../Data/</DataPath>
    E' possibile cambiare questo nei percorsi che seguono, a seconda della piattaforma di lavoro:
        <DataPath>../../Data/</DataPath>
        <DataPath>C:\MyData\</DataPath>
        <DataPath>/home/me/mydata/</DataPath>
