Finestra di Dialogo Livelli

Per aprire la finestra di dialogo livelli, il comando da menu è File: Livelli.
Verranno visualizzati diversi tipi di livelli a seconda se ci si trova sulla Vista Spazio o sulla Vista Terreno.

Livelli sulla Vista Spazio

I livelli sono livelli astratti (dati GIS) disegnati sulla terra come geometrie Punto o Polilinea. La finestra di dialogo mostra unicamente le informazioni sul nome del file ed il numero di elementi di ciascun livello.

Livelli sulla Vista Terreno

Sulla vista terreno è  possibile utilizzare diversi tipi di livelli:

  • Livelli Struttura
  • Livelli Astratti
  • Livelli Immagine
  • Livelli Vegetazione.  Esiste sempre un solo livello vegetazione.  E' possibile Salvare questo nel file dal quale è stato caricato, oppure utilizzare Salvare Come in un altro file.
