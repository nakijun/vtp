Livelli della Vegetazione

Per dettagli su come il software VTP rappresenta la vegetazione, leggete la pagina web VTP Implementation Vegetation. Ciò che segue presuppone che voi abbiate familiarità con la terminologia.

VTBuilder può importare dati vegetazione dai seguenti:

  • ESRI Shape files (.shp)
  • USGS LULC (Land Use / Land Cover, in generale l'enstensione del file é .gir)

Ci sono diversi tipi di dati Vegetazione.

  1. Biotipo - aree poligonali che descrivono il tipo di pianta trovate in ciascuna area.
  2. Densità - aree poligonali che contengono l'informazione (compresa tra 0 e 1) su quanto é desa la vegetazione.
  3. Esplicita - un gruppo di istanze di piante.

Attualmente è possibile:

  • Importare Biotipi da Shapefile.
  • Importare Densità da LULC e Shapefile.
  • Leggere e scrivere istanze Esplicite con file VF, che possono essere letti anche da VTP Enviro

Prima di fare qualunque operazione con la vegetazione,

  1. Siate sicuri di avere un file delle specie necessarie. Un file di esempio é VTP/Enviro/Data/PlantData/species.xml
  2. Caricate il vostro file di specie con il comando da menu "Vegetazione: Lista delle Specie". 

Se avete i dati puntuali di ciascuna pianta, questi sono i passi per importare le istanze esplicite delle piante:

  1. Usate il comando da menu "Livellio: Importa Dati" ed indicate Vegetazione.
  2. Indicate il file che coniene i punti delle vostre istanze.  Generalmente si tratta di uno SHP file.
  3. Salvate il livello risultante in un file VF, che può essere usato da Enviro.

Se avete un'ara coperta da una specie singola di pianta, questi sono i passi per generare una distribuzione di vegetazione:

  1. Usate lo Strumento Area per indicare la zona in cui volete distribuire la vegetazione
  2. Date il comando da menu "Strumento Area - Genera Vegetazione".  Un pannello di dialogo vi chiederà :
    • Spaziatura della Griglia di campionamento, in metri.  La distribuzione avverrà su una griglia regolare.  Per ogni punto, verrà testato quale tipo di pianta ricade in quella posizione, considerando tutti dati di ingresso conosciuti.  Quindi, se viene generata una pianta, viene applicato un piccolo slittamento per evitare un aspetto troppo "regolare" rispetto ad una distribuzione casuale.
    • Scarsità Artificiale.  Molti dei sistemi di simulazione interattiva non possono gestire la densità completa della vegetazione del mondo reale.  Potete impostare questo valore nferiore ad 1 per produrre una distribuzione artificialmente scarsa.  Per esempio, 0.001 produrrà 1/1000 della normale densità di vegetazione.
  3. Il risultante file VF può essere usato da Enviro.

Se invece avete dati poligonali che descrivono l'area generale coperta da un certo numero di piante, usate questi passi per generare una distribuzione di vegetali:

  1. Caricate la vostra Bioregione (lista di Biotipi) con il comando da menu "Vegetazione: BioRegioni".
    Un file di esempio é in Data/PlantData/bioregion.txt
    Le densità sono descritte da piante per metro quadro.
  2. Importate almeno un livello di vegetazione con Biotipo.  Biotipo é un indice che parte da 0, e così un campo intero con valori di 0, 1, 2, 3 ecc. indica quale biotipo.
  3. Se avete anche (optionalmente) dati poligonali con informazioni aggiuntive sulla densità di piante, importate quel livello come Densità.  Questo può essere lo stesso file GIS che contiene il Biotipo, oppure un file separato come un LULC file.  Se non avete un file dati, verrà assunta la "Normale Densità".
  4. Usate lo Strumento Area per indicare l'area su cui distribuire la vegetazione
  5. Date il comando da menu "Strumento Area - Genera Vegetazione".  Si aprirà un pannello di dialogo che chiederà :
    • Spaziatura della Griglia di campionamento, in metri.  La distribuzione avverrà su una griglia regolare.  Per ogni punto, verrà testato per vedere quale tipo di pianta cade in quella posizione, secondo i dati di ingresso conosciuti. Poi, se una pianta é generata, viene applicato un piccolo slittamento per evitare l'aspetto troppo "regolare" di una distribuzione su una griglia.
    • Scarsità Artificiale.  Molti dei sistemi di simulazione interattiva non possono gestire la densità completa della vegetazione del mondo reale.  Potete impostare questo valore nferiore ad 1 per produrre una distribuzione artificialmente scarsa.  Per esempio, 0.001 produrrà 1/1000 della normale densità di vegetazione.
  6. Questo file potrà essere nuovamente caricato in VTBuilder per cercare di correggere la distribuzione, oppure può essere caricato i Enviro per la visualizzazione 3D.
