Panoramica

VTBuilder è uno strumento per visualizzare e processare dati geospaziali.  Si possono importare un'ampia varietà di formati di file, e produrre efficientemente i dati per software per il tempo reale 3D come VTP Enviro.

Il procedimento tipico è il seguente:

  1. Acquisire dati geospaziali grezzi (elevazione, vettoriale delle strade, ecc.)
  2. Caricarli in VTBuilder.
  3. Ripulire i dati con operazioni come l'allineamento, l'estrazione di aree di interesse, l'assemblaggio ed il ricampionamento, e l'aggiunta di informazioni mancanti.
  4. Esportare i dati prodotti.

Concetti

Livelli

VTBuilder tratta i dati come Livelli, in modo simile ad altri strumenti software GIS.
I Livelli sono organizzati nei seguenti tipi:

  • Elevazione
  • Immagini
  • Strutture (Edifici, Recinzioni..)
  • Strade
  • Vegetazione
  • Acqua
  • Grezzi (dati astratti senza correlazione geofisica)

Poichè VTBuilder conosce il tipo di dati contenuti in ogni livello, può eseguire operazioni su di essi che sfruttano la conoscenza geospaziale d'alto livello.

Ogni livello corrisponde ad un file su disco. Il file può essere importato oppure di formato nativo.  I Livelli possono essere letti da un formato importato, e può essere sia letto che scritto in formati nativi.

Progetti

Potete salvare una descrizione di tutti i livelli correnti in un file di progetto (.vtb).  Questa é una maniera conveniente per ricordare i file che sono stati utilizzati, e per riaprirli velocemente tutti più tardi.

Per i dati importati, il file di progetto memorizzerà il nome del file importato, che sarà di nuovo importato quando il progetto sarà aperto.  Per i formati nativi, il livello é semplicemente caricato dal suo file nativo.
