Usage

The VTP software and associated data are entirely free to use, modify, incorporate, redistribute, etc.  There are no restrictions of any kind.  Please read the license and disclaimer for details.

Please send bug reports, feature requests, and all other feedback to Ben Discoe (ben@vterrain.org) or to the VTP Mailing List, but please first read the Software FAQ.

Requirements / Recommendations

  • A fast computer running Windows, Linux, Mac OS X, or other flavor of Unix
  • A reasonably modern 3D card / chipset.
