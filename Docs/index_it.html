Uso

Il software VTP ed i dati associati sono completamente liberi per l'uso, la modifica, l'inclusione, la ridistribuzione, ecc.  Non ci sono restrizioni i qualunque tipo.  Si prega di leggere la licenza e disclaimer per dettagli.

Si prega di inviare report di bug, richieste di nuove feature, e altri suggerimenti a Ben Discoe (ben@vterrain.org) oppure alla VTP Mailing List, ma si prega prima di leggere le Software FAQ.

Requisiti di sistema / Raccomandazioni

  • Un computer veloce con Windows, Linux, Mac OS X, o un altro buon Unix
  • Una scheda 3D ragionevolmente moderna
